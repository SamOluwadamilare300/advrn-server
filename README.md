## 🌟 Overview

**Advrn.ng** is a comprehensive two-sided marketplace designed specifically to assist young professionals relocating for work in major Nigerian cities including **Lagos**, **Abuja**, **Port Harcourt**, and **Kano**. Our platform creates a seamless connection between **tenants, landlords, and employers**, revolutionizing the rental process with flexible payment options, verified user roles, integrated logistics, and employer-secured accommodations.

### 🎯 Mission
To eliminate the stress and uncertainty of finding quality housing for Nigeria's mobile workforce while building trust between all stakeholders in the rental ecosystem.

## 👥 User Roles

### 🧑‍💼 **Tenant (Young Professional)**
**Primary Users:** Recent graduates, job switchers, remote workers, expatriates

**Key Capabilities:**
- Browse verified property listings with detailed information
- Apply filters for budget, location, amenities, and commute time
- Submit rental applications with digital document upload
- Choose from available payment plans set by landlords
- Schedule property viewings and virtual tours
- Access moving and logistics services
- Rate and review properties and landlords
- Track application status and lease progress
- Receive automated reminders for payments and renewals

### 🏠 **Landlord (Property Owner)**
**Primary Users:** Individual property owners, real estate investors, property management companies

**Key Capabilities:**
- List multiple properties with comprehensive details and media
- Set flexible lease terms and preferred payment structures
- Configure property amenities, rules, and tenant requirements  
- Review and manage tenant applications with scoring system
- Opt into insurance plans for property protection
- Access analytics on property performance and market trends
- Communicate directly with prospective and current tenants
- Generate automated lease agreements and receipts
- Track rent payments and manage renewals

### 🏢 **Employer (Corporate Sponsor)**
**Primary Users:** HR departments, startups, multinational companies, consulting firms

**Key Capabilities:**
- Verify corporate credentials (CAC registration)
- Secure housing for employees during relocation
- Manage multiple employee housing arrangements
- Set housing budgets and approval workflows
- Track housing-related expenses and generate reports
- Coordinate with logistics partners for employee moves
- Monitor employee satisfaction with housing arrangements
- Handle lease negotiations on behalf of employees

## 💰 Payment Structures

### 💵 **Payment Options**
| Payment Type | Frequency | Benefits | Ideal For |
|--------------|-----------|----------|-----------|
| **Monthly** | Every 30 days | Lower initial outlay, better cash flow management | Fresh graduates, contract workers |

## 🔐 User Verification

### 🆔 **Tenant Verification**
**Required Documents:**
- Valid government-issued ID (National ID, Driver's License, International Passport)
- Employment verification letter or contract
- Recent payslips or bank statements (last 3 months)
- Previous landlord references (if applicable)

**Optional Enhancements:**
- Employer sponsorship letter
- BVN verification for enhanced trust score
- Professional references

### 🏠 **Landlord Verification** 
**Required Documents:**
- Certificate of Occupancy (C of O) or valid title documents
- Lagos State Resident Registration Agency (LASRRA) registration
- Government-issued personal ID
- Property tax receipts

**Corporate Landlords:**
- Certificate of Incorporation (CAC)
- Tax Identification Number (TIN)
- Property management license (if applicable)

### 🏢 **Employer Verification**
**Required Documents:**
- Certificate of Incorporation (CAC)
- Tax clearance certificate
- Company registration with Corporate Affairs Commission
- Authorized signatory identification
- HR department contact verification

**Verification Process:**
- Automated document validation using OCR technology
- Manual review by verification specialists
- Background checks through government databases
- Reference calls for enhanced verification levels

## 🏗️ Platform Architecture

### 🔧 **Technical Stack**
```
Frontend:     React.js / Next.js with TypeScript
Backend:      Node.js with Express.js / NestJS
Database:     PostgreSQL with Redis caching
Payment:      Paystack, Flutterwave integration
Storage:      AWS S3 / Cloudinary for media
Hosting:      AWS / Google Cloud Platform
Monitoring:   Sentry for error tracking
Analytics:    Google Analytics / Mixpanel
```

### 🏛️ **Architecture Pattern**
- **Modular Monolith:** Organized codebase with clear module boundaries
- **Event-Driven:** Asynchronous processing for notifications and workflows
- **API-First:** RESTful APIs with GraphQL for complex queries
- **Microservice-Ready:** Architected for easy future service separation

### 🔌 **Key Integrations**
- **Payment Gateways:** Secure transaction processing
- **Verification APIs:** Automated document and identity verification
- **Mapping Services:** Google Maps for location and route optimization
- **Communication:** Email, SMS, and push notification services
- **Analytics:** User behavior tracking and business intelligence

## 🚀 Getting Started

### ⚙️ **Prerequisites**
- Node.js 18+ and npm/yarn
- PostgreSQL 12+
- Redis 6+
- AWS/GCP account (for production)

### 📥 **Installation**

```bash
# Clone the repository
git clone https://github.com/relocate-ng/platform.git
cd relocate-ng-platform

# Install dependencies
npm install

# Set up environment variables
cp .env.example .env
# Edit .env with your configuration

# Set up database
npm run db:migrate
npm run db:seed

# Start development server
npm run dev
```

### 🌍 **Environment Variables**
```bash
# Database
DATABASE_URL=postgresql://user:password@localhost:5432/relocate_ng
REDIS_URL=redis://localhost:6379

# Authentication
JWT_SECRET=your-jwt-secret-key
JWT_EXPIRES_IN=7d

# Payment Gateways
PAYSTACK_SECRET_KEY=sk_test_your_paystack_key
FLUTTERWAVE_SECRET_KEY=FLWSECK_TEST-your_flutterwave_key

# File Storage  
AWS_ACCESS_KEY_ID=your_aws_access_key
AWS_SECRET_ACCESS_KEY=your_aws_secret_key
AWS_S3_BUCKET=relocate-ng-uploads

# External Services
GOOGLE_MAPS_API_KEY=your_google_maps_key
SENDGRID_API_KEY=your_sendgrid_key
```
